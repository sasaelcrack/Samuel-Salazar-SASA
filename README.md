# Algoritmos Básicos de Python

## Autor
**Samuel Salazar Nanclares**

## Descripción

Este repositorio contiene una colección de **25 programas en Python** que implementan algoritmos básicos y soluciones a problemas cotidianos de programación. Los ejercicios están diseñados para practicar conceptos fundamentales como entrada/salida de datos, operaciones matemáticas, estructuras condicionales, bucles y formato de salida.

Cada programa está contenido en un archivo independiente y resuelve un problema específico.
